# Tela de Matrícula

Projeto desenvolvido em Python com o objetivo de praticar a criação de interfaces gráficas utilizando o framework Flet.

## Funcionalidades

* Campo para nome do aluno
* Campo para série
* Campo para CPF
* Botão de envio
* Captura dos dados digitados pelo usuário

## Tecnologias

* Python 3.11
* Flet

## Objetivo

Este projeto foi criado para estudar os componentes básicos do Flet, como textos, campos de entrada, botões e eventos.

## Aprendizados

Durante o desenvolvimento foram praticados os seguintes conceitos:

* Criação de interfaces gráficas
* Manipulação de eventos
* Captura de dados de formulários
* Organização de componentes em uma aplicação Flet

## Melhorias Futuras

* Validar os dados informados
